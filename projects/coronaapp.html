CoronaApp

Made with PHP, MySQL and Java for Android

During the first Lockdown back in March 2020, I've read about the Development of a Smartphone App used for tracking Contact Persons using BlueTooth. Because it took quite a While until the official App was released, together with a Friend I've worked on our own Version which ended up to be very similar to the Original one.

This Project was developed between April and May 2020.
