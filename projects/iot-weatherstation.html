IoT-capable Weatherstation

Made with Arduino / C++, PHP and MySQL

The goal of this Project was to develop a Weatherstation capable to push its Measurements into a Database, from where the Data could be pulled to be used on any other Device. You can have a Look at my Code on the GitHub Repository.

This Project was developed between November 2019 and March 2020.
