3D-Printer

Made with C++ and MarlinFirmware

This is a custom-designed FDM 3D Printer running on Marlin Firmware. It has a Print Size of approximately 14x14x20cm. All the Models are uploaded on the GitHub Repository.

This Project was developed between November 2020 and March 2021.
